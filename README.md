## 📋 About

A modern, professional portfolio website showcasing Ashish Kumar Sankhua's journey as an AI Product Consultant at Salesforce India. This portfolio demonstrates expertise in product strategy, execution, Salesforce platforms, and AI/ML technologies with a clean, responsive design and smooth user interactions.
